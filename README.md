# Ethiopia Financial Inclusion Forecasting

Forecasting account ownership and digital payment adoption in Ethiopia (2025-2027).

## Business Context
Selam Analytics engaged by consortium including NBE, mobile money operators, and development finance institutions.

## Data
- Global Findex surveys (2011-2024)
- Ethiopia-specific events (Telebirr, M-Pesa, policy changes)
- Unified schema with observations, events, impact_links, and targets

## Setup
pip install -r requirements.txt

## Structure
- data/raw/ - Original datasets
- data/data_enrichment_log.md - Documentation of additions
- notebooks/ - Analysis code
- reports/ - Visualizations
- dashboard/ - Streamlit app
